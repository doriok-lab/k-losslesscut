# K-LosslessCut
<p>K-LosslessCut은 동영상 자르기 프로그램입니다.
<p>FFmpeg에 기반하여 다양한 포맷의 동영상을 효율적으로 처리할 수 있습니다.
<br>특히, 직접 스트림 복사(Direct Stream Copy) 방식을 이용하면 엄청난 속도의 퍼포먼스를 보여줍니다.  
<br>사용자 편의를 고려하여 GUI 환경을 제공합니다.
