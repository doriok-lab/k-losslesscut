LUFS

1. LUFS의 뜻

- LUFS는 방송 텔레비전 시스템과 기타 동영상 및 음악 스트리밍 서비스에서 오디오 정규화에 사용되는 표준 음량 측정 단위입니다.

- 인간의 귀가 소리를 인식하는 방식과 유사한 방식으로 오디오 프로그램 라우드니스를 측정하는 알고리즘이 적용됩니다.

2. 표준 LUFS를 권장하는 이유

- 유튜브의 경우, 업로드된 동영상의 음량이 -14LUFS를 넘을 때에는 정규화를 통해 해당 동영상의 음량을 줄여버립니다. 다음과 같은 목적에서입니다.

- 일관성 유지: 시청자들이 동일한 채널의 여러 영상을 시청할 때, 소리의 크기가 일정하게 유지되어야 함. LUFS를 기준으로 하면 다양한 영상 간에 일관성 있는 음량을 제공할 수 있음.

- 다이나믹 레인지 조절: LUFS를 사용하여 노멀라이즈하면 다이나믹 레인지가 줄어들 수 있음. 다이나믹 레인지를 줄이는 것은 음질에 나쁜 영향을 미칠 수 있으므로 적절한 레벨을 유지하는 것이 중요함.

- 시청자 편의성: 시청자들이 영상을 시청할 때 소리가 너무 크거나 작아서 불편을 느끼지 않도록 하기 위해 LUFS를 기준으로 함.

3. 표준 LUFS를 초과했을 때

- 유트브의 경우를 볼까요? 동영상 위에 마우스 커서를 놓고 오른쪽 버튼을 누르면 메뉴 중에 '전문통계'가 보일 겁니다. 이것을 클릭하면 다음과 같은 팝업창이 뜹니다.

여기서 'Volume / Normalized' 줄을 보세요.


▲ (표준 LUFS를 초과한 경우) -7.5LUFS로 업로드된 경우 Volume / Normalized 100% / 47% => 정규화된 결과 볼륨이 줄여져 스트리밍된다는 뜻. content loudness 6.5dB => 업로드 LUFS와 표준 LUFS의 편차


▲ (표준 LUFS에 맞춘 경우) -14LUFS로 업로드된 경우 Volume / Normalized 100% / 100% => 업로드된 볼륨 그대로 스트리밍된다는 뜻. content loudness 0.0dB => 업로드 LUFS와 표준 LUFS의 편차

- 만약, -14LUFS보다 낮게 업로드하면 스트리밍 시 음량이 커지는 건가요? 그렇지는 않습니다. 이 경우에는 업로드된 음량 그대로 스트리밍됩니다.

4. 스트리밍 서비스별 표준 LUFS

LUFS 측정 / 볼륨 조정

1. LUFS 목표치 설정

- '설정' 메뉴에서 LUFS 목표치를 설정합니다.
기본값 : 유튜브 등 스트림 서비스의 LUFS 목표 수준인 -14LUFS


▲ (예시) LUFS 목표치를 -14로 설정.

2. LUFS 측정과 볼륨 조정

- '설정'에서 정한 LUFS 목표치와 실제 측정된 LUFS의 차이를 구해 볼륨을 증감합니다.
볼륨 증감폭은 자동으로 계산됩니다.

- 볼륨을 조정한 후 다시 LUFS를 측정합니다. 사용자가 볼륨 조정을 멈출 때까지 반복합니다.

- LUFS 목표치와 실제 측정된 LUFS가 같아졌을 때 반복 작업은 멈춥니다.
